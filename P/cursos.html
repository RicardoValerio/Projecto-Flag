Web Programmer & Mobile

Descrição

Programação para a Web e Mobile está associada ao conceito de desenvolvimento de sites e aplicações para a web e mobile usando as linguagens de programação mais utilizadas no mercado. O Web e Mobile Developer é o profissional criativo responsável pelo desenvolvimento de sites dinâmicos, portais e aplicações destinadas tanto para ambiente web como mobile sendo atualmente uma das profissões mais procuradas no mercado de trabalho.

Objetivos

Dotar os formandos de todos os conhecimentos teóricos e competências necessários para ingressar no mercado de trabalho e assumir uma atividade profissional ligada à Programação Web e Mobile: programação de sites para web e mobile, desenvolvimento e manutenção de plataformas de gestão de conteúdo, programação de aplicações para a web, criação de aplicações mobile para Android e iOS, entre outros.

Destinatários

Todos os interessados em iniciar ou consolidar uma atividade profissional como Programador Web e Webmobile.

Graphic Designer

Descrição

Design Gráfico é uma das atividades do foro criativo mais disseminadas no mercado atual, sendo o processo criativo e técnico que recorre a imagens e texto para comunicar, informar, identificar, sinalizar, organizar, estimular, persuadir e entreter.

Objetivos

Dotar os formandos de todos os conhecimentos teóricos e competências necessários para ingressar no mercado de trabalho e assumir uma atividade profissional ligada ao Design Gráfico: pós-produção fotográfica, tratamento digital de imagens, design de sistemas de identidade visual, design de embalagens ou sinalética, design editorial e paginação, publicidade, documentos interativos e microsites, entre outras.

Web Design

Descrição

Web e Mobile Design são extensões do design cujo foco é a criação de websites e documentos disponíveis na internet. Tratam-se de domínios multidisciplinares, na medida em que implicam competências de diversas áreas, para além do design propriamente dito, tais como arquitetura de informação, programação, usabilidade, e criação de interfaces para mobile, entre outros.

Objetivos

Dotar os formandos de todos os conhecimentos teóricos e competências necessários para ingressar no mercado de trabalho e assumir uma atividade profissional ligada ao Web e Mobile Design: desenvolvimento de sites e documentos para mobile, desenho de interfaces para a web e dispositivos móveis, criação de animações em flash, entre outros.
